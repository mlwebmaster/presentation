Description

The Datum DPL5+ Cross Line Laser produces two bright red vertical and horizontal laser line beams for alignment work, as well as 5 spot beams for setting out 90 Degree angles easily and quickly.

These spot beams can be used in conjunction with the horizontal and vertical line alignment beams if required.

With a built-in compensator, the DPL5+ accurately and quickly self levels automatically for operation with a single button and all functions easily selectable using only two buttons.

The DPL5+ is a Pulse Beam instrument that can be used externally with the Datum Pulse Laser Detector included in the kit for a working range as far as 120m.
